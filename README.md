| This project was made during Hacaton at Salt. Web scrapper of NTS radio shows.

### Stack: 

- ReactJS 17.0.x
- Styled-components 5.2.x
- React-spring 8.0.x

## Setting up your dev environment

To install locally follow the directions in the MongoDB documentation.

`npm install`

To run server

`npm start`
